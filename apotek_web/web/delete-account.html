Permintaan Penghapusan Akun Medika

Jika Anda memutuskan untuk berhenti menggunakan aplikasi Medika, Anda dapat mengajukan permintaan penghapusan akun Anda beserta seluruh data pribadi terkait (nama, email, transaksi, dan riwayat resep) secara permanen dari sistem kami.

Penting: Proses penghapusan data bersifat permanen dan tidak dapat dibatalkan. Setelah akun dihapus, Anda tidak akan dapat login kembali atau memulihkan riwayat transaksi Anda.
✓ Permintaan Anda telah terkirim! Tim support kami akan memverifikasi dan menghapus seluruh data Anda dalam waktu maksimal 3x24 jam. Konfirmasi penghapusan akan dikirimkan ke email Anda.
